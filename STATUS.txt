 TROUBLESHOOTING

"Cannot connect to backend"
  → docker compose up -d
  → docker compose ps

"Ollama not running"
  → On Windows host: ollama run llama2
  → Don't close the PowerShell window

"No results found"
  → Try different phrasing
  → Check generated SQL in UI
  → Verify database has data

"Slow responses"
  → Normal: 2-5 seconds for Llama
  → Ensure 4GB+ memory available
  → Check: docker logs selcom-backend -f
